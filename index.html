Know which models your key can actually use

Usable LLM checks real model access for OpenAI-compatible and Anthropic-compatible endpoints. It tests each listed model with a live request and explains whether it is usable, blocked, or skipped.

Enter your API details and click Check Models to get started.

How it works

Most providers will list models your account can see, but that does not always mean your key can actually use them. Usable LLM verifies access by sending a small live request to each model.

01

Discover

We read the provider’s /v1/models endpoint and collect the models your API key is allowed to view.

02

Verify

We send a minimal test request to each supported model using the correct protocol for that provider.

03

Explain

Usable means the model responded. Blocked means the provider rejected the request. Skipped means it is not a chat or messages model.
