How it works

Unlike tools that just list models from /v1/models, we actually test each one.

1

Fetch model list

Queries /v1/models to discover every available model on your endpoint.

2

Test each model

Sends a real max_tokens=1 request to every chat-capable model. If it responds, you can use it.

3

See results

Instantly see which models are ✅ usable, ❌ blocked, or ⏭ skipped — with error details for blocked ones.
